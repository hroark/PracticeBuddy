## Practice Buddy
This is a tool to help guitar (and other instrument) players create and organize their practice routines.

**Planned features:**

 - Set routine timers
 - Allow routines to be moved to different order
 - "Swap Meet" to share routines (possible gamification)
 - Notes
 - Category of routine
 - Ability to "Stash" routines for later use
 - Ability to delete routines
 - Collection of routines can be organized into buckets (days of the week, for instance"
 - Ability to provide links to resources

**Want to have:**

 - Mobile versions
 - Metronome that allows user to set stepping of clicks
 - Anything else that comes to mind!
